pratyu
======

<h1>RAD 2013</h1>

<p>Como rodar o "localhost/pratyu":</p>
<ul>
<li>Instalar o composer : http://getcomposer.org/download/</li>
<li>Para instalar escolher o php.exe em .../wamp/bin/php/phpX.X.XX/php.exe</li>
<li>Ativar as extensões php->php extensions->php_openssl e apache->apache modules->ssl_module</li>
<li>Na pasta .../wamp/bin/php/phpX.X.XX/ abrir o arquivo php.ini e descomentar
	a linha ;extension=php_openssl.dll (tirar o ; )</li>
<li>Exercutar o Git Shell e dentro da pasta wamp/www/pratyu/ executar o comando composer install</li>
